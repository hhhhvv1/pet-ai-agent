# 🐾 Pet AI Agent (2026.03 ~ 2026.06)

반려동물의 건강 상태를 분석하고 기록을 관리할 수 있는 AI 기반 헬스케어 에이전트입니다.

## 📌 프로젝트 소개

Pet AI Agent는 보호자가 반려동물의 증상을 입력하면 AI가 위험도를 분석하고 적절한 조치를 제안하는 프로그램입니다.

또한 건강 기록을 저장하고, 건강 변화 추적, 예방 건강 체크, 병원 방문 준비 및 사후 관리 기능을 제공합니다.

---

## ✨ 주요 기능

### 1. AI 건강 분석

* 반려동물 정보 기반 증상 분석
* 위험도 평가
* 권장 행동 제안

### 2. 건강 기록 저장

* 분석 결과 자동 저장
* SQLite 데이터베이스 활용

### 3. 건강 기록 조회

* 과거 건강 기록 확인
* 증상 및 위험도 조회

### 4. 응급 상황 감지

다음과 같은 증상 발견 시 응급 상황으로 판단합니다.

* 경련
* 의식 없음
* 호흡곤란
* 혈변
* 피를 토함
* 쓰러짐

### 5. 예방 건강 체크

* 식사 상태 확인
* 수분 섭취 확인
* 배변 상태 확인
* 활동량 확인

### 6. 병원 방문 준비

* 증상 발생 시점 기록
* 복용 약 정보 정리
* 과거 병력 정리

### 7. 병원 방문 후 관리

* 진단명 기록
* 처방약 관리
* 회복 상태 점검

### 8. 건강 변화 추적

* 최근 건강 기록 분석
* 반복 증상 확인
* 건강 추세 파악

### 9. PDF 건강 리포트 생성

* 건강 기록 PDF 저장
* 병원 제출용 자료 활용 가능

## 🚀 실행 방법

### 1. 패키지 설치

```bash
pip install openai
pip install reportlab
```

### 2. 데이터베이스 생성

```bash
python3 database.py
```

### 3. LM Studio 실행

* LM Studio 설치
* Gemma 모델 로드
* Local Server 실행

### 4. 프로그램 실행

```bash
python3 main.py
```
